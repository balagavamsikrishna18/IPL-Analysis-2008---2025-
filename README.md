# IPL-Analysis (2008 - 2025)
## Objective 
The objective of this projct is to create intertive Power BI dashbboard that provides insights into the 
performance of reqiriment IPL season from 2008 to 2025. 
A key business requirment is to enable user to select any season using slicer/filter annd innstantly view:

## Data set used 
Ipl data 2008 - 2025 [click to view](https://github.com/balagavamsikrishna18/IPL-Analysis-2008---2025-/blob/main/ipl_matches_data.xlsx)

## Questions (KPIs Covered)
- Total Sixes in each season 
- Total Fours in each Season
- Total Teams Participated 
- Total Matches Played i Season 
- Total centuries in each Season 
- Total Half-Centuries  in each Season
- Total Venues Used in each Season
